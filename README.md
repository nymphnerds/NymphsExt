# Nymphs3D2 Extensions

This repo is a static Blender extension repository for `Nymphs3D2`.

It contains:

- `index.json`
- `index.html`
- packaged extension zip files

Use this repository from Blender's Extensions preferences by adding the raw `index.json` URL for this repo.

Example pattern:

```text
https://raw.githubusercontent.com/Babyjawz/Nymphs3D2-Extensions/main/index.json
```
